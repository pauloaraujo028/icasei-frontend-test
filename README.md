## Sobre o Projeto

Este projeto consiste em uma aplicação frontend modularizada usando micro-frontends para atender a um teste técnico. A aplicação inclui uma barra lateral de navegação (mf_drawer) e uma página de vídeos (mf_videos).

<img width="1493" alt="nextjs-portfolio" src="https://github.com/pauloaraujo028/icasei-frontend-test/assets/105018299/221adaf5-b708-468c-a392-b2c6f07ca574">
<img width="1493" alt="nextjs-portfolio" src="https://github.com/pauloaraujo028/icasei-frontend-test/assets/105018299/3efe141f-4d99-4dd4-a98b-12c3f6103ae9">

**Funcionalidades principais:**
* Barra lateral de navegação com links para páginas de vídeos e favoritos.
* Busca de vídeos utilizando a API do YouTube.
* Adição de vídeos aos favoritos.
* Proxy reverso com Nginx para resolver problemas de CORS.
* Inicio de testes unitarios

Este projeto consiste em um sistema de gerenciamento de vídeos que inclui uma sidebar (MF_DRAWER) para navegação e uma área de conteúdo principal (MF_VIDEOS) para listar vídeos e favoritos. O sistema foi desenvolvido seguindo as melhores práticas de desenvolvimento front-end e utilizando tecnologias modernas.

## Getting Started

<a name="clone-locally"></a>
### Clonar-localmente
- Digite essa url: [https://github.com/pauloaraujo028/icasei-frontend-test](https://github.com/pauloaraujo028/icasei-frontend-test) no seu navegador.
- Uma vez aberto, navegue até o nível superior esquerdo do projeto, um botão verde de download do código estará visível no lado direito.
- Selecione a opção de download Zip no menu suspenso.
- Quando o download estiver concluído, você poderá acessar meu projeto localmente.
   ou
  ```sh
  git clone https://github.com/seu-usuario/icasei-frontend-test.git
  cd icasei-frontend-test
  ```


<a name="prerequisites"></a>
### Pré-requisitos
* npm
  ```sh
  npm install npm@latest -g
  ```

## Contribuição

As contribuições são o que tornam a comunidade de código aberto um lugar incrível para aprender, inspirar e criar. Quaisquer contribuições que você fizer são **muito apreciadas**.

Se você tiver uma sugestão para melhorar o projeto, faça um fork do repositório e crie uma solicitação pull. Você também pode simplesmente abrir um issue com a tag "melhoria".
Não se esqueça de dar uma estrela ao projeto! Obrigado novamente!

1. Faça o Fork do Projeto
2. Crie seu Branch (`git checkout -b feature/RecursoIncrivel`)
3. Commit suas alterações (`git commit -m 'Adicione algum recurso incrivel'`)
4. Push para o Branch (`git push origin feature/RecursoIncrivel`)
5. Abra um Pull Request
